Lexicon


Worked as a member of the GitHub team in Machine Learning @ Berkeley to improve Linguist, GitHub's file and repository language classifier. With our new dataset and Linear SVC model, we were able to achieve a 40x faster classification time, a 15% increase in accuracy, and we were able to cover 50 additional languages - all without the use of any hard-coded rules! Next, we plan on using RNNs to generate nonlinear features to feed into our Linear SVC to further improve classification.
