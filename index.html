Work

"Have you seen Los Angeles at 4am?" - Kobe Bryant

Here are some of the things I've been working on outside of school:

PubMed Extraction 6/17-Now

ResultCare
Collaborating with the ResultCare data team to develop a classifier that can identify diagnoses from lab tests in medical research papers. Developed a PubMed extraction API that has retrieved 11,000 lab test related articles from a set of ~10 million (all papers published on PubMed since 2007). Currently, I have TF-IDF vectorized the texts and am testing our multi-featured data on linear models.

Deep Active Learning 1-6/17

ML@Berkeley
Team of five researching active learning, a supervised learning framework in which a model minimizes the number / determines the best unlabeled data points to query to improve classification. We hypothesize that this would not only make the data scraping process less expensive, but also result in a model more effective in learning its data. Currently, we’re conducting tests on the Yelp dataset.

Lexicon 8-12/16

ML@Berkeley
Worked as a member of the GitHub team to improve Linguist, GitHub's file and repository language classifier. With our new dataset and Linear SVC model, we were able to achieve a 40x faster classification time, a 15% increase in accuracy, and we were able to cover 50 additional languages - all without the use of any hard-coded rules! Next, we plan on using RNNs to generate nonlinear features to feed into our Linear SVC to further improve classification. Presented our work at GitHub HQ for their engineering all-hands.

bSafe 11/16

CalHacks 3.0
In lieu of the recent outbreak in crimes around student housing at Berkeley, my team designed an app to pool students walking alone at night. Our pooling algorithm matches students based on their origin and destination – finding the minimal area between their “walking” vectors.
